# repl
`b <board string>` - creates a new board with a given string

`l<row>+<count>`    - moves the row to the left by count

`d<col>+<count>`    - moves the column down by count

`r`                 - creates a random board

`p`                 - prints the board

`t`                 - creates the table for hashing

`h`                 - hashes the board

`q`                 - quits
